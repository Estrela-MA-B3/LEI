area :: Figura -> Double #

esta função calcula as áreas de um circulo, quadrado ou rectangulo

type Ponto = (Double, Double) data Figura = Circulo Ponto Double | Rectangulo Ponto Ponto | Quadrado Ponto Double deriving Show

area :: Figura -> Double
area (Circulo (x, y) r) = pi*r^2
area (Quadrado (x, y) l) = l*l
area (Rectangulo (a, b) (c, d)) = (abs (c - a) * abs (d - b))

Exemplos de Utilização:

Expand
>>> area (Circulo (0, 0) 1)
3.14159
